# Cheat820

A collection of browser-console / userscript "cheats" and trainers — **one folder per game**.
Everything here is for personal/offline tinkering and learning how these apps manage their
state. Each tool is **leaderboard-safe / self-only by default** so nothing you do is submitted
or affects other players.

| Folder | Game | What it is |
| --- | --- | --- |
| [`admin82/`](admin82/) | **82-0** (NBA React game) | Console engine + optional floating GUI to max stats, reveal/unlock content, and rewrite your record to a perfect 82-0. Leaderboard writes blocked by default. |
| [`blooket/`](blooket/) | **Blooket** | Self-only client-side trainer userscript (Tampermonkey / Violentmonkey). Affects only your own client/account/screen. |
| [`rngdle/`](rngdle/) | **RNGdle** | Console cheat + floating GUI panel to roll any number, collect every badge, and re-roll past the daily limit. Leaderboard-safe by default. |

See each folder's own `README.md` for full install/usage instructions.
